## Objective

Migrate all modals to Shadcn Dialog with Stitch-consistent styling. Replace all remaining `blue-600` with `#4848e5`. Ensure nav/header uses primary color. Final consistency pass.

## Tasks

<task id="1">
### Migrate Modals to Shadcn Dialog

Convert all `{open && <div className="fixed inset-0...">}` modals to Shadcn Dialog:
- Use `<Dialog open={open} onOpenChange={onClose}>`
- `<DialogContent>` / `<DialogHeader>` / `<DialogTitle>` / `<DialogFooter>`
- Form inputs: `h-11 px-4 rounded-lg border-slate-200 text-sm focus:border-[#4848e5] focus:ring-[#4848e5]` (match register page input style)
- Labels: `text-sm font-medium text-slate-700 mb-1.5`
- Primary button: `bg-[#4848e5] hover:bg-[#4848e5]/90 text-white rounded-lg h-10 px-4 text-sm font-semibold`
- Cancel button: `border border-slate-300 text-slate-700 rounded-lg h-10 px-4 text-sm font-semibold hover:bg-slate-50`
- Destructive button: `bg-red-600 hover:bg-red-700 text-white` for delete actions

Replace `window.confirm()` for delete with Shadcn AlertDialog.
</task>

<task id="2">
### Global Color Migration

Search and replace all remaining `blue-600`, `blue-700`, `blue-50`, `blue-500` references in frontend src:
- `bg-blue-600` → `bg-[#4848e5]`
- `hover:bg-blue-700` → `hover:bg-[#4848e5]/90`
- `text-blue-600` → `text-[#4848e5]`
- `text-blue-700` → `text-[#4848e5]`
- `bg-blue-50` → `bg-[#4848e5]/10`
- `ring-blue-500` → `ring-[#4848e5]/50`
- `border-blue-600` → `border-[#4848e5]`
- Loading spinner: `border-[#4848e5] border-t-transparent`

Update `(app)/layout.tsx` nav link active state to use `bg-[#4848e5]/10 text-[#4848e5]`.
</task>

**Final check:** Verify `npm run build` passes. No remaining blue-600 references.
