Backend endpoints (all require Authorization: Bearer {token} header):
- GET /users → returns User[]
- POST /users → body: { fullName, email, password, role, managerId? } → returns User
- PATCH /users/:id → body: { role?, managerId? } → returns User
- PATCH /users/:id/status → body: { isActive: boolean } → returns User

From frontend/src/lib/api/auth.ts:
```typescript
export function getStoredToken(): string | null
export interface AuthUser {
  id: string; email: string; full_name: string; role: string; company_id: string;
}
export function getStoredUser(): AuthUser | null
```

User shape returned by GET /users:
```typescript
interface User {
  id: string;
  company_id: string;
  full_name: string;
  email: string;
  role: 'owner' | 'admin' | 'manager' | 'employee' | 'executive';
  is_active: boolean;
  manager_id: string | null;
  created_at: string;
}
```
